We provide a training dataset (~10 000 words paired with hypernyms) sampled from the current version of the ruWordNet taxonomy. The test data is annotated manually according to the current snippet of ruWordNet and is split into two parts: public (763 nouns and 175 verbs) and private (1 525 nouns and 350 verbs).


Both training and testing datasets use the same ruWordNet synsets and are annotated in a similar way. They consist of three columns: word, hypernym id (synset id in ruWordNet) and hypernym as word (optional). Each synset of the predicted 10 should be written in a separate line. E.g.:

утка → 4839-N → сельскохозяйственная птица

утка → 5349-N → водоплавающая птица

утка→...→...

утка→...→ …

…

Please use tabs as column separators and save your solutions in the .tsv format. While submitting results please mention additional resources in the comments. The public test set for the practice phases may be found here. The list of private test set words will be provided by the beginning of Evaluation phases.
